# Card Deck

Learning the basics of Go by creating a simple card deck.

#### Functionality includes:

- **newDeck** -> creates a new deck
- **print** -> prints list of all cards in deck
- **shuffle** -> shuffles cards
- **deal** -> create a "hand" of cards
- **saveToFile** -> saves deck to local machine
- **newDeckFromFile** -> load deck from local machine

Following along a [Udemy course](https://www.udemy.com/go-the-complete-developers-guide/) created by Stephen Grider
